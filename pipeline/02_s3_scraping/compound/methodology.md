# Methodology

**Logic designed by Amazon Q, verified by Shannon Goddard.**

## Compound Genetics Extraction

### Minimal Extraction Pipeline
1. **Description** - Product description from collapsible__content
2. **Agronomic Traits** - Lineage, Potency, Flower Time, Veg Time, Aroma
3. **Strain Name** - From URL

### Data Fields
- `genetics_lineage` - Parent strains
- `thc_min` / `thc_max` - Potency range
- `flowering_time` - Weeks to harvest
- `vegetative_time` - Veg period
- `aroma` - Terpene profile
