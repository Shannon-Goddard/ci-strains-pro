# Project Vision: Cannabis Intelligence (CI)

## The Partnership
- **Lead Architect:** Shannon Goddard (Domain expertise, manual cleaning, strategic lead).
- **Technical Assistant:** Amazon Q (Code generation, regex optimization, documentation).

## The Mission
To transform 15,700+ raw records into a tiered, validated "Intelligence" ecosystem. 
We prioritize **Transparency** above all else. Every cleaning step must be documented.

## Data Tiers
1. **Raw**: 15k+ master list.
2. **Unique**: 10k de-duplicated.
3. **Augmented**: 10k with logical assumptions (clearly flagged).
4. **Gold**: 5k true-sourced only.
